### Pronunciation section:

**a gennaio ci sono 2 m quindi la n è lunga → gennaio**
→ In “gennaio” there are two “n/m sounds,” so the “n” is long → gennaio

**a febbraio ci sono due b quindi la b è lunga → febbraio**
→ In “febbraio” there are two “b’s,” so the “b” is long → febbraio

**marzo: la z si pronuncia come “ts” → marzo**
→ In “marzo,” the “z” is pronounced like “ts” → marzo

**aprile: bisogna pronunciare bene tutte le vocali → aprile**
→ In “aprile,” you must pronounce all the vowels clearly → aprile

**maggio: ci sono due g quindi la g è lunga → maggio**
→ In “maggio,” there are two “g’s,” so the “g” is long → maggio

**giugno: gn si pronuncia come “ny” (like in lasagne)**
→ In “giugno,” “gn” is pronounced like “ny” → giugno

**luglio: gl si pronuncia con la lingua sul palato**
→ In “luglio,” “gl” is pronounced with the tongue against the palate → luglio

**agosto: inizia con la “a” → agosto**
→ “Agosto” starts with “a” → agosto

**settembre: ci sono due t quindi la t è lunga → settembre**
→ In “settembre,” there are two “t’s,” so the “t” is long → settembre

**ottobre: anche qui due t → ottobre**
→ In “ottobre,” also two “t’s” → ottobre

**novembre: facile → novembre**
→ “Novembre” is easy → novembre

**dicembre: attenzione alla “i” → dicembre**
→ In “dicembre,” pay attention to the “i” sound → dicembre

# **Section 3: Concepts Explained**

### 1. **Months in Italian**

Months are:

* not capitalized (unlike English)
* used with preposition **“a”**

  * a gennaio → in January

---

### 2. **Italian Pronunciation Rules**

* **Double consonants matter**

  * febbraio (bb) → longer sound
  * settembre (tt)

* **Special sounds**

  * **gn** → like “ny” (giugno)
  * **gl** → soft “lli” (luglio)
  * **z** → “ts” (marzo)

---

### 3. **Cultural Notes**

* **Agosto (August)**
  → Many businesses close → vacation culture

* **Settembre (September)**
  → School starts

* **Aprile proverb**
  → “Aprile dolce dormire” (spring = sleepiness 😄)
